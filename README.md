# 💖 Valentine's Day Website

A fun and interactive **Valentine's Day-themed webpage** built using **HTML**, **CSS**, and **JavaScript**. This project was created to practice front-end development by designing an engaging, playful experience with animations and interactive elements.

## 🚀 Live Demo

👉 https://anshuchourasia.github.io/valentines-day/

## ✨ Features

- 💖 Romantic Valentine's Day theme
- 🎨 Attractive and colorful user interface
- 🖱️ Interactive buttons and animations
- 📱 Responsive design for desktop and mobile
- ⚡ Smooth user experience
- 🌐 Lightweight and fast-loading webpage

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
